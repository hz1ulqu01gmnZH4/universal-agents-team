## Part 1: Architecture Overview

### What Phase 9 Adds

Phase 9 closes the gap between library code and runnable system. The framework has
92 source files across 8 phases, all individually tested (1491 tests), but nothing
wires them together. Phase 9 adds:

1. **Config generation** — bootstrap creates all 10 missing engine config files
2. **FrameworkFactory** — single-call factory that constructs a fully-wired Orchestrator
3. **Missing CLI tools** — `review-submit`, `team-manager`, `topology-analyze`, `rehash-constitution`
4. **Updated CLAUDE.md generator** — orchestration instructions for Claude Code agents

### What Phase 9 Does NOT Include

- **Autonomous run loop** — Claude Code IS the runtime; the framework is a library
- **Real agent process management** — agents are Claude Code Tasks, not subprocesses
- **Message passing** — Phase 10 scope (inter-agent messaging protocol)
- **Meta-evolution** — evolution engine evolving itself

### Key Principles

1. **One-call construction**: `FrameworkFactory.build(root, domain)` returns a working
   Orchestrator with all dependencies wired. No manual assembly required.
2. **Phase-gated construction**: Factory only constructs engines for phases whose
   config files exist. Missing config = phase disabled, not crash.
3. **Fail-loud on bad config**: If a config file exists but has wrong structure,
   KeyError propagates immediately (IFM-N53). No silent degradation.
4. **CLAUDE.md as driver**: The generated CLAUDE.md contains step-by-step
   orchestration instructions. Claude Code reads these and drives the framework.

## Part 3: FrameworkFactory (`src/uagents/engine/framework_factory.py`)

### 3.1 Purpose

Single-call factory that constructs a fully-wired Orchestrator with all dependencies
resolved in correct order. Phase-gated: only constructs engines whose config files
exist under `core/`.

### 3.2 Implementation

```python
"""Framework factory — one-call Orchestrator construction.
Spec reference: Phase 9 (Integration & Wiring).

Constructs all engine dependencies in dependency order and returns
a fully-wired Orchestrator. Phase-gated: missing config files disable
the corresponding phase, they don't crash.
"""
from __future__ import annotations

import logging
from pathlib import Path

import yaml

from ..audit.logger import AuditLogger
from ..state.directory import DirectoryManager
from ..state.yaml_store import YamlStore

logger = logging.getLogger("uagents.factory")


class FrameworkFactory:
    """Constructs a fully-wired Orchestrator from a bootstrapped framework root.

    Usage:
        factory = FrameworkFactory(root=Path("/path/to/framework"), domain="meta")
        orchestrator = factory.build()
    """

    # Config file → phase mapping for phase-gating
    PHASE_CONFIGS: dict[str, str] = {
        "core/resource-awareness.yaml": "1.5",
        "core/environment-awareness.yaml": "2.5",
        "core/canary-expectations.yaml": "2.5",
        "core/skill-system.yaml": "3",
        "core/context-pressure.yaml": "3.5",
        "core/tool-taxonomy.yaml": "3.5",
        "core/evolution.yaml": "4",
        "core/self-governance.yaml": "5",
        "core/creativity.yaml": "6",
        "core/scout.yaml": "7",
    }

    def __init__(self, root: Path, domain: str = "meta") -> None:
        self.root = root.resolve()
        self.domain = domain
        self._enabled_phases: set[str] = set()
        self._detect_phases()

    def _detect_phases(self) -> None:
        """Detect which phases are enabled by checking for config files."""
        for config_path, phase in self.PHASE_CONFIGS.items():
            full_path = self.root / config_path
            if full_path.exists() and full_path.stat().st_size > 0:
                self._enabled_phases.add(phase)
                logger.info("Phase %s enabled (found %s)", phase, config_path)
            else:
                logger.info("Phase %s disabled (missing %s)", phase, config_path)

    def _phase_enabled(self, phase: str) -> bool:
        return phase in self._enabled_phases

    def build(self) -> "Orchestrator":
        """Construct and return a fully-wired Orchestrator.

        Construction order follows the dependency tree:
        Tier 0: YamlStore, AuditLogger, paths
        Tier 1: TaskLifecycle, ResourceTracker
        Tier 2: BudgetTracker, RateLimiter, CostGate (Phase 1.5)
        Tier 3: PromptComposer, AgentSpawner
        Tier 4: TopologyRouter, ReviewEngine, TeamManager
        Tier 5: Self-awareness engines (Phase 2)
        Tier 6: Environment engines (Phase 2.5)
        Tier 7: Skill engines (Phase 3)
        Tier 8: Protection engines (Phase 3.5)
        Tier 9: Evolution & governance (Phases 4-5)
        Tier 10: Creativity & expansion (Phases 6-7)
        Tier 11: Population evolution (Phase 8)
        Tier 12: Orchestrator assembly
        """
        # ── Tier 0: Foundation ──
        yaml_store = YamlStore(self.root)
        instance_root = self.root / "instances" / self.domain
        log_root = instance_root / "logs"
        audit_logger = AuditLogger(log_root)
        constitution_path = self.root / "CONSTITUTION.md"
        state_dir = instance_root / "state"

        # ── Tier 1: Core state ──
        from .task_lifecycle import TaskLifecycle
        from .resource_tracker import ResourceTracker

        task_lifecycle = TaskLifecycle(yaml_store, self.domain)
        # ResourceTracker created after budget/rate (Tier 2) so we can inject them

        # ── Tier 2: Budget & resource (Phase 1.5) ──
        budget_tracker = None
        rate_limiter = None
        cost_gate = None

        if self._phase_enabled("1.5"):
            from .budget_tracker import BudgetTracker
            from .rate_limiter import RateLimiter
            from .cost_gate import CostGate

            # FM-P9-032/033 fix: Read config values and pass to constructors
            ra_config = yaml_store.read_raw("core/resource-awareness.yaml")
            ra = ra_config["resource_awareness"]

            budget_tracker = BudgetTracker(
                yaml_store, self.domain,
                plan=ra["claude_max_plan"],
            )
            rl_cfg = ra["rate_limits"]
            rate_limiter = RateLimiter(
                yaml_store, self.domain,
                rpm_estimate=int(rl_cfg["rpm_estimate"]),
                itpm_estimate=int(rl_cfg["itpm_estimate"]),
                otpm_estimate=int(rl_cfg["otpm_estimate"]),
            )
```

### 3.3 Design Decisions

1. **Phase-gated, not all-or-nothing**: Missing config = phase disabled. This lets
   users start with Phase 0 only and progressively enable phases by adding config
   files. No crash on missing config, but KeyError on malformed config.

2. **Lazy imports**: Each tier imports only when constructing that tier. This avoids
   import-time errors from phases that aren't enabled.

3. **Direct attribute wiring for Orchestrator internals**: `diversity_engine` and
   `stagnation_detector` are set via direct attribute assignment because the Orchestrator
   constructor stores them as `self.diversity_engine` and `self.stagnation_detector`
   (not constructor parameters — they were added in Phase 2 after the constructor was
   defined).

4. **Single GitOps instance**: All evolution components share one GitOps to prevent
   concurrent git operations from different instances.

```python
"""Rehash constitution after legitimate human edits.
Spec reference: Axiom A3 (Constitutional Immutability)."""
from __future__ import annotations

import argparse
import hashlib
from pathlib import Path


def main() -> None:
    parser = argparse.ArgumentParser(
        description="Recompute constitution hash after legitimate edits"
    )
    parser.add_argument("--root", default=".", help="Framework root directory")
    args = parser.parse_args()

    root = Path(args.root).resolve()
    constitution_path = root / "CONSTITUTION.md"
    hash_path = root / "core" / "constitution-hash.txt"

    if not constitution_path.exists():
        print("Error: CONSTITUTION.md not found", file=__import__("sys").stderr)
        raise SystemExit(1)

    content = constitution_path.read_text(encoding="utf-8")
    new_hash = hashlib.sha256(content.encode("utf-8")).hexdigest()

    old_hash = ""
    if hash_path.exists():
        old_hash = hash_path.read_text(encoding="utf-8").strip()

    hash_path.write_text(new_hash, encoding="utf-8")

    if old_hash and old_hash != new_hash:
        print(f"Constitution hash updated:")
        print(f"  Old: {old_hash[:16]}...")
        print(f"  New: {new_hash[:16]}...")
    else:
        print(f"Constitution hash: {new_hash[:16]}...")


if __name__ == "__main__":
    main()
```
